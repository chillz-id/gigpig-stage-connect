# Show Active Plans Status

List all active implementation plans and their progress.

## Check Locations

1. **Project Plans:** `/root/agents/Plans/`
2. **Session Plans:** `/root/.claude/plans/`

## For Each Plan, Report:
- File name and date
- Goal/purpose
- Completion status (count checked vs unchecked items)
- Next action needed

## Action
Search both plan directories, read each plan file, and summarize the status of all active work.
