# Verification Before Completion

Before marking this task as complete, verify the following:

## Verification Checklist

### 1. Code Quality
- [ ] Run `npm run lint` - Does it pass?
- [ ] Run `npm run build` - Does it compile without errors?

### 2. Testing
- [ ] Are there tests for the new/changed functionality?
- [ ] Run `npm run test` - Do all tests pass?
- [ ] For UI changes: Did you manually test in browser?

### 3. Functionality
- [ ] Does the feature work as intended?
- [ ] Did you test edge cases?
- [ ] Are there any console errors?

### 4. Documentation
- [ ] Is the plan file updated with completion status?
- [ ] Are any necessary comments added to complex code?

## If Any Check Fails
Do not claim the task is complete. Instead:
1. Note what failed
2. Fix the issue
3. Re-run the checks

## If All Checks Pass
- Update the plan file to mark the task complete
- Summarize what was accomplished
- Note any follow-up items

---

Run the verification commands and report the results.
